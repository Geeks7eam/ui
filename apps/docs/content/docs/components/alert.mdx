# Alert

The `Alert` in web design is a highlighted box that grabs attention, often used for brief messages or important information.

## Basic Alert

To create a basic alert, you can simple use it like example in below:

<Example>
    <Preview>
        <Alert
            title='Alert Title'
            icon={<SparkleIcon size={20} />}
            description={
                <span className='text-inherit'>
                    You can pass <code  className='!text-inherit'>ReactNode</code> as description.
                </span>
            }
        />
        <hr className='my-4' />
        <Alert
            title='Alert Title (Column Direction)'
            direction='column'
            icon={<SparkleIcon size={32} />}
            description={
                <span className='text-inherit'>
                    You can pass <code  className='!text-inherit'>ReactNode</code> as description.
                </span>
            }
        />
    </Preview>

    ```tsx showLineNumbers
    import Alert from '@zyxui/alert';

    <Alert
        title='Alert Title'
        icon={<SparkleIcon size={20} />}
        description={
            <span>
                You can pass <code>ReactNode</code> as description.
            </span>
        }
    />
    <hr />
    <Alert
        title='Alert Title (Column Direction)'
        direction='column' // set direction of content to column
        icon={<SparkleIcon size={32} />}
        description={
            <span className='text-inherit'>
                You can pass <code>ReactNode</code> as description.
            </span>
        }
    />
    ```
</Example>

## API Reference

For an in-depth exploration of available props and advanced usage, consult the `Alert` component's API reference:

| Property            | Description                               | Type        | Default     |
| --------------------|-------------------------------------------|-------------|-------------|
| icon | To add icon for alert. | `ReactNode` | `null` |
| title | Alert title. | `ReactNode` | `null` |
| description | Alert description. | `ReactNode` | `null` |
| className | Adding custom class names to your alert. | `string` | `undefined` |
| color | Customize alert color styles. | `default` \| `error` \| `warning` \| `success` | `default` |
| variant | Change alert base styles. | `solid` \| `bordered` \| `flat` \| `shadow` \| `gradient` | `solid` |
| radius | Change alert border radius style. | `sm` \| `md` \| `lg` \| `circle` \| `none` | `md` |
| direction | Change alert `flex-direction`. | `row` \| `column` | `row` |
