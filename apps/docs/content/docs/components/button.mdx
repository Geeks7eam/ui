# Button

The `Button` element in web development is an HTML element used to create interactive buttons that users can click or tap to trigger specific actions or functions on a web page. The `<button>` element is an essential part of web forms, user interfaces, and general interactivity.

## Basic Button

You can use a basic button without passing any props to `button` component, thus it will take primary color as default.

<Example>
  <Preview className='flex flex-row gap-10 justify-center items-center'>
    <Button>Basic</Button>
  </Preview>

  ```jsx showLineNumbers
  import {Button} from 'ui';

  <Button>Basic</Button>
  ```

</Example>



## Props

We have such of props that make you able to customize button as you like. Below you will find more examples about how to use this props, and what each one of them is going to do.

### Colors

Our `Button` component have many color options. Which are `default` | `error` | `warning` | `success`. Each one of these values is referencing to a color. If you want to change these default colors check [`How to customize themes in @Geeks/UI`](#)

<Example>
  <Preview className='flex flex-row gap-10 justify-center items-center'>
    <Button color='default'>Default</Button>
    <Button color='error'>Error</Button>
    <Button color='warning'>Warning</Button>
    <Button color='success'>Success</Button>
  </Preview>

  ```jsx showLineNumbers
  import {Button} from 'ui';

  <Button color='default'>Default</Button>
  <Button color='error'>Error</Button>
  <Button color='warning'>Warning</Button>
  <Button color='success'>Success</Button>
  ```

</Example>




### Variants

You can change the style of button by playing with `variant` prop. Which can accept follow values: `solid` | `bordered` | `light` | `ghost` | `flat` | `shadow`

<Example>
  <Preview className='flex flex-row gap-10 justify-center items-center'>
    <Button variant='solid'>Solid</Button>
    <Button variant='bordered'>Bordered</Button>
    <Button variant='light'>Light</Button>
    <Button variant='ghost'>Ghost</Button>
    <Button variant='flat'>Flat</Button>
    <Button variant='shadow'>Shadow</Button>
  </Preview>

  ```jsx showLineNumbers
  import {Button} from 'ui';

  <Button variant='solid'>Solid</Button>
  <Button variant='bordered'>Bordered</Button>
  <Button variant='light'>Light</Button>
  <Button variant='ghost'>Ghost</Button>
  <Button variant='flat'>Flat</Button>
  <Button variant='shadow'>Shadow</Button>
  ```

</Example>


### Sizes

You can change size of your buttons based on this prop. Which allows you control the size of your `Button` component by passing one of these values: `sm` (Small) | `md` (Medium) | `lg` (Large)

<Example>
  <Preview className='flex flex-row gap-10 justify-center items-center'>
    <Button size='sm'>Small</Button>
    <Button size='md'>Medium</Button>
    <Button size='lg'>Large</Button>
  </Preview>

  ```jsx showLineNumbers
  import {Button} from 'ui';

  <Button size='sm'>Small</Button>
  <Button size='md'>Medium</Button>
  <Button size='lg'>Large</Button>
  ```

</Example>


### Radiuses

Also you are able to change `border-radius` of your button with `radius` props: `sm` | `md` | `lg` | `circle` | `none`.
