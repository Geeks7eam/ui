import Text from '@zyxui/text'
import Button from '@zyxui/button'

    const [direction, setDirection] = useState('row')
    return (
        <Flex direction='column' gap='lg'>

                <Button size='sm' variant={direction === 'column-reverse' ? 'flat' : null} onPress={() => {setDirection('column-reverse')}}>column-reverse</Button>
            </Flex>
            <Flex direction={direction} gap='md'>
                <div className='bg-pink-800 px-10 py-2'>01</div>
                <div className='bg-pink-800 px-10 py-2'>02</div>
                <div className='bg-pink-800 px-10 py-2'>03</div>
                <div className='bg-pink-800 px-10 py-2'>04</div>
                <div className='bg-pink-800 px-10 py-2'>05</div>
            </Flex>
        </Flex>
    )
}

# Flex Component

The `Flex` component provides a flexible and powerful way to manage layouts in your React applications. With a rich set of variants and customization options, it allows you to easily control the direction, alignment, and spacing of child elements.



## Installation

Ensure that have the @zyxui library installed in your project:
<Example>
```bash
npm install @zyxui/flex
```
</Example>


## Basic Usage

Import the `Flex` component and use it to create flexible layouts:


<Example>
    <Preview>
        <Flex className='bg-cyan-400/20'>
            <div className='px-10 py-2 bg-cyan-400 text-white font-medium'>01</div>
            <div className='px-10 py-2 bg-cyan-400 text-white font-medium'>02</div>
            <div className='px-10 py-2 bg-cyan-400 text-white font-medium'>03</div>
            <div className='px-10 py-2 bg-cyan-400 text-white font-medium'>04</div>
        </Flex>
    </Preview>
    ```tsx showLineNumbers
    import Flex from '@zyxui/flex'

    const FlexExample = () => {
        return (
            <Flex>
                <div>01</div>
                <div>02</div>
                <div>03</div>
                <div>04</div>
            </Flex>
        )
    }

    export default FlexExample
    ```
</Example>


## Customization

Tailor the behavior of the `Flex` component to suit your layout needs with the following customization options.

### Direction

The `direction` prop allows you to control the direction of `flex` container. This porp can have many values: `column` | `column-reverse` | `row` | `row-reverse`, and the default value is: `row`.

<Example>
    <Preview>
        <DynamicFlexDirection />
    </Preview>

## API Reference

For an in-depth exploration of available props and advanced usage, consult the `Alert` component's API reference:

| Property            | Description                               | Type        | Default     |
| --------------------|-------------------------------------------|-------------|-------------|
| as | Component element. | `h1` \| `h2` \| `h3` \| `h4` \| `h5` \| `h6` \| `p` \| `span` \| `mark` \| `code` | `h1` |
| color | Customize text color. | `default` \| `error` \| `warning` \| `success` | `default` |
| className | Adding custom class names to your text element. | `string` | `undefined` |
| weight | Change text font-weight value. | `thin` \| `extralight` \| `light` \| `normal` \| `medium` \| `semibold` \| `bold` \| `extrabold` \| `2extrabold` | `normal` |
