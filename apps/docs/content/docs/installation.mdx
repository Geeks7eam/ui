# Installation

Welcome to **@zyxui**, your go-to React UI components library for creating stunning, high-performance user interfaces effortlessly.

## How to Use @zyxui Components

Let's kick things off with the powerful **<a href='/docs/components/button'>Button</a>** Component. Get ready to dive into the world of @zyxui with these simple steps:

### 1.Install via npm

Install **@zyxui** components into your project using the following command in your terminal:

<Example style={{ margin: 0 }}>

```bash showLineNumbers
npm install @zyxui/config @zyxui/button
```
</Example>

### 3.Set up TailwindCSS Config File

Locate the `tailwind.config.js` file in your project's root directory and meticulously apply the following adjustments:

<Example>
```javascript showLineNumbers {5, 7}
const zyxuiConfigs = require('@zyxui/config/tailwind.config');

/** @type {import('tailwindcss').Config} */
module.exports = {
  presets: [zyxuiConfigs],
  content: [
    ...zyxuiConfigs.content,
    './pages/**/*.{js,ts,jsx,tsx}',
    './components/**/*.{js,ts,jsx,tsx}',
    './app/**/*.{js,ts,jsx,tsx}',
    './src/**/*.{js,ts,jsx,tsx}',
  ],
  plugins: [require('@tailwindcss/typography')],
};

```
</Example>
